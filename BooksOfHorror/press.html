Description

Books of Horror is a procedural horror game. Each run mixes stories with different mechanics and rules in a random environment, making every game unpredictable, both in terms of gameplay and the level generated.

History

Created by Antonio Manzano under the Kuro Kosmico label, the project started as an exploration of procedural systems. With over 4 years of experience in Unity, Antonio aimed to build a "Mechanical Bible" for horror, where the environment itself feels like a sentient, threatening entity.

Features

  • The Living Maze: Rooms generated tile-by-tile, not just by pre-fab rooms.
  • Retro Aesthetic: A visual style that evokes classic horror with modern lighting.
  • Multiple Mecanics Systems: Mechanics mixed to make diferent experiences in each run
